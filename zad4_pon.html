1) Kiedy powstał JavaScript (podaj datę)
JavaScript powstał w 1995 roku.

2) Dlaczego jest to język skryptowy
Ponieważ jest interpretowany w trakcie działania programu, a nie kompilowany przed uruchomieniem.

3) Gdzie jest wykonywany JS (klient lub serwer)
JavaScript jest wykonywany głównie po stronie klienta (przeglądarka), ale może być też po stronie serwera (np. Node.js).

4) Dlaczego JS to bezpieczne narzędzie
Ponieważ działa w tzw. piaskownicy (sandbox), nie ma bezpośredniego dostępu do plików użytkownika i systemu operacyjnego.

5) Czy jest to język obiektowy, jeśli tak to dlaczego
Tak, ponieważ obsługuje obiekty, dziedziczenie i metody.

6) Wymień zalety JS:
Zaleta1: Jest uniwersalny i działa na wszystkich przeglądarkach.
Zaleta2: Pozwala tworzyć dynamiczne strony internetowe.
Zaleta3: Jest łatwy do nauki i posiada dużą społeczność.

7) Framework to:
Gotowy szkielet do tworzenia aplikacji, który przyspiesza pracę programisty.

8) Do aplikacji mobilnych można użyć Framework
Tak, istnieją frameworki do aplikacji mobilnych, np. React Native.

9) Aplikacje responsywne to:
Aplikacje, które dostosowują się do różnych rozmiarów ekranu i urządzeń.

10) Przepisz linie kody pod nimi wytłumaczenie jak działa ten fragment kodu
‹input type="button" name="przycisk" value="Nowa Strona" onclick="WinOpen(' ')"›
Tworzy przycisk, który po kliknięciu wywołuje funkcję otwierającą nowe okno lub stronę.

11) Przepisz linie kody pod nimi wytłumaczenie jak działa ta ten fragment kodu
window.open("obraz.html","okienko","toolbar=no,directories=no,menubar=no,height=280,width=160,top=200,left=200");
Otwiera nowe okno przeglądarki z plikiem obraz.html i ustawia jego właściwości.
toolbar=no → ukrywa pasek narzędzi
directories=no → ukrywa przyciski katalogów
s menubar=no → ukrywa menu
height=280 → wysokość okna 280 pikseli
width=160 → szerokość okna 160 pikseli
top=200 → pozycja okna 200 pikseli od góry ekranu
left=200 → pozycja okna 200 pikseli od lewej strony ekranu

12) Opisz następującą instrukcję window.close()
Zamykana jest bieżąca karta lub okno przeglądarki, w którym została wywołana ta funkcja.
